CSS ‚Äãfont-family and ‚Äãfont properties allow a comma-separated list of fonts, but React Native styles fontFamily allows only one font name. With the ‚ÄãTRE, you can instruct which fonts are available in the system (or fonts you have added), and it will pick the first match! The prop to achieve that is ‚ÄãsystemFonts.
